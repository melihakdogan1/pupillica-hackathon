💊 İlaç Asistanı

TİTCK İlaç ve Tıbbi Ürün Bilgi Sistemi
468,406 doküman içinde gelişmiş AI arama
⚠️ Önemli Uyarı: Bu sistem yalnızca bilgi amaçlıdır. İlaç kullanımı ile ilgili kararlarınızı almadan önce mutlaka doktorunuza danışın. TİTCK verilerinden oluşturulmuş akademik araştırma projesidir.

🚀 Canlı Demo Sistemi

İlaç Asistanı, TİTCK'nin 4,816 PDF dokümanından çıkarılan 468,406 metin parçası üzerinde gelişmiş AI-powered semantic search gerçekleştiren bir sistemdir.

📍 Demo için: Proje sahipleri ile iletişime geçerek canlı sistemin gösterimini talep edebilirsiniz.

🔍

Semantic Search

Doğal dil işleme ile anlamsal arama. "ateş düşürücü", "yan etki", "doz bilgisi" gibi terimlerle arama yapabilir.

⚡

Hızlı Performans

468,406 doküman içinde 250-700ms'de sonuç. ChromaDB vector database ile optimize edilmiş arama.

📱

Responsive Tasarım

Mobil, tablet ve masaüstü cihazlarda mükemmel çalışan modern web arayüzü.

🛡️

Güvenli API

Rate limiting, CORS koruması ve güvenli veri işleme ile korunmuş sistem.

📊

Kapsamlı Veri

TİTCK'nin tüm ilaç ve tıbbi ürün belgelerini kapsayan geniş veri tabanı.

🤖

AI Teknolojisi

Sentence Transformers modeli ile gelişmiş doğal dil anlama ve benzerlik skoru hesaplama.

🛠️ Teknoloji Stack

Backend
FastAPI, Python 3.11
AI/ML
Sentence Transformers, all-MiniLM-L6-v2
Database
ChromaDB Vector Database
Frontend
Modern HTML5, CSS3, JavaScript
Data Processing
PyPDF2, pdfplumber, multiprocessing
Deployment
GitHub Pages, Local Demo
